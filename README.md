# CCTV_Digest

把中国《新闻联播》当作**信号**来读的一台私人投研工具：每天自动总结、解读，并从长周期梳理出中国发展的几条"主线"，做成以**信号河流（River）**为骨架、**大字编辑刊（Editorial）**为阅读页的可视化网站。

> 核心理念：联播的价值不在"事实"，而在**编辑选择层**（上场顺序、时长、提法/措辞变化、连续敲鼓）。产品把这份"宣传"转化为*带时间维度、带置信度的投资假设*（政策主题 → 行业 → 示例标的），并始终区分【内容】/【信号】/【推断】。

## 功能

- **每日简报 + 雷达**：当天最重要的信号 + "今日什么在变"（新提法/口径翻转/敲鼓升降）。
- **脉络河流**：2009→今，发展主线随时间此消彼长的全屏流图；点某天动效进入当日解读。
- **探索**：关键词/提法时间序列、行业敲鼓热力矩阵、提法首现时间线。
- **系统管理**（鉴权）：按流水线环节配置 LLM 供应商与模型。

## 技术

Next.js (App Router) · Neon Postgres · Vercel Blob · Vercel Cron · 多供应商 LLM 抽象（默认 DeepSeek V4，可切换 Vercel AI Gateway / DeepSeek 官方 / 硅基流动 / 其他 OpenAI 兼容）。早期全用免费档。
